# GrepCoin

## The AI-Powered Play-to-Earn Crypto Arcade on Base

GrepCoin (GREP) is a deflationary token powering an ecosystem of browser games, staking rewards, and achievement NFTs. Built on Base L2 for low-cost, fast transactions.

## Platform Features

### Play-to-Earn Arcade
- 9 browser-based games
- Earn GREP for every game played
- Staking multipliers boost earnings
- Anti-cheat protection

### Real Yield Staking
- 4 tiers: Basic, Silver, Gold, Diamond
- Rewards from platform revenue (not inflation)
- Lock periods: 0 to 90 days
- Bonus daily plays

### Achievement NFTs
- Soulbound badges for accomplishments
- Mint on-chain as proof
- Unlock through gameplay
- Burn GREP to mint

### Deflationary Burns
- Marketplace fees (2.5%)
- Tournament entries (10%)
- Achievement mints (10 GREP)
- Guild creation (500 GREP)
